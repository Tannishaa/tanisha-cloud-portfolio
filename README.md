##  Featured Projects

### [Flux-Order: High-Concurrency Order Engine](https://github.com/Tannishaa/flux-order)
* **Problem:** Preventing overselling and race conditions during "Flash Sale" traffic spikes.
* **Solution:** Decoupled architecture using **AWS SQS** and **Redis Distributed Locks (Mutex)**.
* **Tech:** Python, Redis, AWS (SQS, DynamoDB), Terraform (IaC). 

### [Automated Expense Audit & ETL Pipeline](https://github.com/Tannishaa/bill-e-audit)
* **Problem:** Manually auditing financial records for fraud is slow and error-prone.
* **Solution:** A serverless ETL pipeline that triggers on S3 uploads and uses a Heuristic Risk Engine to flag anomalies. 
* **Tech:** AWS Lambda, S3, Amazon Athena, Pandas.
